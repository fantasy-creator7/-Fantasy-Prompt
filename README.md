# Fantasy Prompt Generator

**Agasa Noble Art** presents a fantasy prompt generator designed to help you create prompts for AI image generation. With its intuitive interface, you can quickly generate prompts for high-quality fantasy artwork and line art suitable for coloring pages.

## How to Use

1. Open `index.html` in your web browser.
2. Select your preferred options using the on-screen buttons (such as setting, character, background, and more).
3. Copy the generated prompt by clicking the **Copy** button.
4. Paste the prompt into your preferred AI image generation platform (such as Google Flow) and generate your image.
